# 🌟 Random Quote App

A modern Random Quote Generator built using HTML, CSS, and JavaScript.

## ✨ Features

- 🎲 Random inspirational quotes
- 📋 Copy quote to clipboard
- 🌙 Automatic Dark Mode support
- 📱 Fully Responsive Design
- 🎨 Clean and Minimal UI
- ⚡ Smooth animations
- 🏷️ Quote categories/tags

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
